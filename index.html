How it works

  • Click or tap on the rectangle above to create a tone and a circle.
  • The rectangle is divided into 6 sections, each with different frequencies and colors.

About

  • DUBloom is an HTML5 Canvas application inspired by the Bloom iOS app from Brian Eno and Peter Chilvers.
  • 100% JavaScript! This page downloads no images or audio files.
  • Code available on github.com at https://github.com/dubharmonic/DUBloom
  • The visuals and audio will work on the latest versions of all popular desktop browsers. Currently mobile browsers only support the visuals.
